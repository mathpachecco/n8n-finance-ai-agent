# 💰 AI Financial Agent for WhatsApp

An AI-powered financial assistant built with n8n that helps users manage their personal finances through WhatsApp.

## Features

- 📲 WhatsApp integration
- 🤖 AI-powered conversations
- 💵 Register income and expenses
- 📊 Monthly financial summaries
- 📈 Expense analysis by category
- 📝 Automatic Google Sheets updates
- 🧠 Conversation memory with Redis
- ⚡ Built with n8n low-code automation

## Technologies

- n8n
- WhatsApp
- Evolution API
- Google Sheets
- Redis
- OpenRouter
- Google Gemma
- AI Agent

## Workflow

1. User sends a message on WhatsApp.
2. The AI understands the financial transaction.
3. Data is validated.
4. The transaction is stored in Google Sheets.
5. The assistant confirms the operation.
6. Financial reports can be generated on demand.

## Example

User:

```
I spent R$50 on groceries today.
```

Assistant:

```
Your expense has been recorded successfully.

Category: Food
Amount: R$50.00
```
